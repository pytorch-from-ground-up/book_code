# Contributing

Thanks for helping improve the companion notebooks.

## Report a problem

Open an issue and include:

- chapter number and notebook path;
- the cell or section where the problem appears;
- Python and PyTorch versions;
- operating system and device (CPU, CUDA, or MPS);
- the complete error message and the smallest reproduction you can provide.

## Propose a change

Keep examples CPU-friendly, deterministic where possible, and free of required external downloads. Preserve the chapter's scope and avoid copying book pages or long passages into the repository. Run `python scripts/validate_notebooks.py` before submitting a pull request.
